Two Equivalent Patterns

When working with an if block, you can use the boolean expression directly in the if statement, or you can assign the expression to a variable and then use that variable in the condition. Both approaches have the same effect.

Pattern 1: Direct Boolean Expression

Place the boolean expression directly in the if statement:

x = 10 if x > 5 and x < 20: print("x is between 5 and 20")

Pattern 2: Boolean Variable

First assign the boolean expression to a variable, then use that variable in the if statement:

x = 10 is_in_range = x > 5 and x < 20 if is_in_range: print("x is between 5 and 20")

When to Use Each Pattern

Use Pattern 1 (direct expression) for simple conditions. It is concise and readable:

x = 5 if x == 5: print("x equals 5") if x > 0 and x < 100: print("x is in range")

Use Pattern 2 (boolean variable) when:

  • The condition is complex or long, and giving it a descriptive name makes the code clearer
  • You want to reuse the same condition in multiple places
  • You want to explain what the condition checks by using a meaningful variable name
age = 25 has_valid_id = age >= 18 and age <= 65 has_clean_record = True if has_valid_id and has_clean_record: print("Eligible")

Both patterns are valid — choose the one that makes your code clearest.
